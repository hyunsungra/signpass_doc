# QR코드를 이용한 계약서 작성

QR코드를 활용하여 현장에서 즉시 계약서를 작성하고 서명받는 방법을 안내합니다.

## QR코드 계약서 개요

### 활용 시나리오
- **오프라인 매장**: 고객이 매장 방문 시 즉시 계약
- **행사장**: 박람회, 세미나에서 즉석 계약
- **방문 영업**: 태블릿/스마트폰으로 현장 계약
- **키오스크**: 무인 계약 시스템
- **대면 상담**: 상담 후 즉시 계약 진행

### 주요 장점
- 종이 문서 불필요
- 즉시 계약 가능
- 고객 편의성 증대
- 계약 관리 자동화
- 비대면 계약 가능

## QR코드 생성하기

### 1. 계약서 준비
#### 템플릿 선택 또는 생성
- 자주 사용하는 계약서 템플릿 준비
- 고정 내용은 미리 입력
- 가변 항목만 입력 필드로 설정

#### QR 전용 계약서 설정
- 모바일 최적화 레이아웃
- 간소화된 입력 필드
- 터치 친화적 UI

### 2. QR코드 생성
#### 생성 방법
1. 계약서 편집 완료 후 '공유' 메뉴 선택
2. 'QR코드 생성' 클릭
3. QR코드 옵션 설정
4. QR코드 다운로드

#### QR코드 옵션
- **유효 기간**: 1일, 7일, 30일, 무제한
- **사용 횟수**: 1회용, 다회용, 무제한
- **접근 제한**: 비밀번호, 지역 제한
- **자동 만료**: 특정 날짜/시간

### 3. QR코드 커스터마이징
#### 디자인 옵션
- QR코드 색상 변경
- 로고 삽입
- 프레임 추가
- 크기 조절

#### 정보 포함
- 계약서 제목
- 회사명
- 안내 문구
- 유효 기간 표시

### 1. 인쇄물 활용

## 고객 계약 프로세스

### 1. QR코드 스캔
#### 스캔 방법
- 스마트폰 카메라 앱
- QR코드 리더 앱
- 카카오톡 QR 스캔

#### 접속 확인
- 계약서 페이지 로딩
- 모바일 최적화 확인
- 네트워크 연결 상태

### 2. 계약서 작성
#### 정보 입력
- 고객 정보 입력
  - 이름, 연락처
  - 주소, 이메일
- 계약 조건 선택
  - 상품/서비스 옵션
  - 계약 기간
  - 결제 방법

#### 본인 인증
- 휴대폰 인증
- 간편 인증 (PIN)
- 생체 인증 (지문, 얼굴)

### 3. 전자서명
#### 서명 진행
- 터치스크린 직접 서명
- 서명 패드 제공
- 저장된 서명 사용

#### 서명 완료
- 계약서 즉시 발송
- 고객 이메일로 사본 전송
- 계약 완료 안내

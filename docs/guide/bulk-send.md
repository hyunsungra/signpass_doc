# 계약서 대량 발송

여러 명의 수신자에게 동시에 계약서를 발송하는 대량 발송 기능을 안내합니다.

## 대량 발송 개요

### 활용 시나리오
- **인사 관리**: 전 직원 근로계약서 갱신
- **프랜차이즈**: 가맹점 일괄 계약
- **교육 기관**: 수강생 동의서 수집
- **이벤트**: 참가자 동의서 일괄 발송
- **회원 관리**: 약관 변경 동의

### 주요 기능
- 최대 1,000명 동시 발송
- 개인별 맞춤 정보 자동 입력
- 일괄 추적 및 관리
- 자동화된 리마인더
- 통합 리포트 제공

## 대량 발송 준비

### 1. 계약서 템플릿 준비
#### 변수 설정
- `{{이름}}`: 수신자 이름
- `{{회사명}}`: 소속 회사
- `{{직급}}`: 직급/직책
- `{{계약금액}}`: 개인별 금액
- `{{시작일}}`: 계약 시작일

#### 변수 삽입 방법
1. 계약서 편집 시 변수 삽입 버튼 클릭
2. 원하는 변수 선택 또는 새로 생성
3. 계약서 내 적절한 위치에 배치

### 2. 수신자 목록 준비
#### Excel 파일 형식
```
| 이름 | 이메일 | 휴대폰 | 회사명 | 직급 | 계약금액 |
|------|--------|---------|---------|------|-----------|
| 홍길동 | hong@email.com | 010-1234-5678 | A회사 | 과장 | 5,000,000 |
| 김철수 | kim@email.com | 010-2345-6789 | B회사 | 대리 | 4,500,000 |
```

#### CSV 파일 형식
```csv
이름,이메일,휴대폰,회사명,직급,계약금액
홍길동,hong@email.com,010-1234-5678,A회사,과장,5000000
김철수,kim@email.com,010-2345-6789,B회사,대리,4500000
```

### 3. 데이터 검증
#### 필수 항목 확인
- 이메일 주소 유효성
- 휴대폰 번호 형식
- 필수 변수 값 존재 여부

#### 중복 확인
- 이메일 중복 제거
- 동일 수신자 확인
- 데이터 정합성 검사

## 대량 발송 실행

### 1. 발송 마법사 시작
1. 대시보드에서 '대량 발송' 메뉴 선택
2. 준비한 계약서 템플릿 선택
3. '대량 발송 시작' 클릭

### 2. 수신자 목록 업로드
#### 파일 업로드
- Excel(.xlsx, .xls) 또는 CSV 파일 선택
- 드래그 앤 드롭으로 업로드
- 최대 파일 크기: 10MB

#### 데이터 매핑
- 업로드된 컬럼과 변수 매핑
- 자동 매핑 또는 수동 설정
- 미리보기로 확인

### 3. 발송 옵션 설정
#### 발송 방식
- **즉시 발송**: 바로 전송
- **예약 발송**: 특정 시간 지정
- **순차 발송**: 일정 간격으로 발송

#### 발송 채널
- 이메일 (기본)
- SMS (추가 옵션)
- 카카오톡 (추가 옵션)
- 복합 채널 (이메일 + SMS)

#### 기타 옵션
- 서명 마감일 설정
- 자동 리마인더 설정
- 발송 실패 시 재시도

### 4. 발송 전 최종 확인
#### 미리보기
- 무작위 수신자 샘플 확인
- 변수 치환 결과 검토
- 레이아웃 확인

#### 테스트 발송
- 관리자에게 테스트 발송
- 실제 수신 상태 확인
- 링크 작동 여부 테스트

## 발송 모니터링

### 1. 실시간 대시보드
#### 발송 현황
- 총 발송 건수
- 성공/실패 건수
- 진행률 표시

#### 수신자 상태
- **발송 완료**: 성공적으로 전송됨
- **발송 실패**: 이메일/SMS 전송 실패
- **열람함**: 계약서를 확인함
- **서명 완료**: 서명을 완료함
- **거부**: 서명을 거부함

### 2. 상세 추적
#### 개별 수신자 추적
- 발송 시간
- 열람 시간 및 횟수
- 서명 진행 상황
- IP 주소 및 디바이스 정보

#### 필터링 및 검색
- 상태별 필터링
- 이름/이메일 검색
- 날짜 범위 설정
- 커스텀 필터

### 3. 자동 알림
#### 관리자 알림
- 발송 완료 알림
- 오류 발생 알림
- 일일 진행 리포트

#### 수신자 리마인더
- 미열람자 자동 재발송
- 서명 마감 임박 알림
- 맞춤형 독촉 메시지

## 후속 조치

### 1. 미완료 건 관리

#### 자동 저장

- 자동 백업 설정

### 3. 보고서 생성
#### 발송 결과 리포트
- 전체 요약 통계
- 수신자별 상세 내역
- 시간대별 분석
- Excel/PDF 내보내기

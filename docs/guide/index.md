# SIGNPASS 서비스 소개

SIGNPASS는 전자계약 서비스로, 복잡한 계약 과정을 간소화하여 언제 어디서나 PC와 모바일에서 손쉽게 계약을 체결할 수 있는 플랫폼입니다. 특히 근로계약서 작성에 특화되어 있으며, 법적 효력이 보장되는 전자서명을 통해 안전하게 계약을 관리할 수 있습니다.

## 주요기능
1. 계약서 관리
2. 계약서 발송 및 작성
3. 계약서 QR 작성
4. 계약서 대량 발송
5. 출퇴근 관리
6. 서명 관리
